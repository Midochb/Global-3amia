PATCH 72 — Insta-ready word cards (social style)

Files changed:
- style.css
- mot/index.html
- mot/mot.js

What it adds:
- A subtle “share header” on each word page: flag + cultural icon + dialect label
- Country-based color accents (TN, MA, DZ, EG, LB, IQ, SD + default)
- Screenshot / capture mode: click the 📸 button (top-right of the card) to hide most UI chrome and get a clean mobile-friendly capture

How to install:
1) Copy/replace the 3 files in your repo with the ones from this zip (keep the same paths)
2) Deploy on Netlify

Tips:
- For clean screenshots, open any word page and click the 📸 button
- You can also force capture mode by adding ?capture=1 to the URL
